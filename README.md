Baan ~ A collection of posts.
A secure CRUD (Create, Read, Update, Delete) web application using Laravel and MySQL, implementing Laravel Breeze for user authentication and access control. The application allows authenticated users secure access with registration, login, logout, password hashing, and session-based authentication. Utilized Laravel's MVC architecture, Eloquent ORM, routing, controllers, and Blade templates to build a structured and maintainable application. Integrated database operations to ensure data integrity and reliability.
